Workshop Requirements & Details

AI Dev Jumpstart requires:

  • GitHub Copilot license
  • Admin rights on your computer
  • ZScaler Developer Exceptions group access for NodeJS

AI-assisted coding is becoming mainstream at an incredible pace, and the developer transformation on use of these tools is here today. AI Dev Jumpstart is BCS program for hands-on leveling from developer to developer in small groups, investing our AI learning hours into this group learning activity.

Prerequisites: GitHub Copilot -license, ready to use on your choice of IDE. Admin rights. Ability to install node (Zscaler developer exceptions group).

We use GitHub Copilot Agentic mode on greenfield development of learning applications in pair and ensemble format. That is, zero slides. Rotating control on what we are doing on each participant.

GitHub Copilot of today is a different tool it was before. GitHub Copilot CLI is a fairly recent addition. So is Claude Opus 4.6. And four level context controls with prompts, project level instructions, agent and skills files. And awesome-copilot skillshare as well as CGI AI exchange.

It does not matter if you use GitHub Copilot, Claude Code, Codex, Kiro, Cursor or Lovable. Except that for GitHub Copilot enterprise governance features, this one is likely to make its way to our day to day in scale first. The tools change, and we need to have experiences on how they help us.

While we don’t want to push you to recent comparison of 13x improvement in time to a result, the numbers are starting to look like we just should not ignore this for our professional developer repertoire.

The world does not need more vibe coded applications. Developers don’t need the work of fixing vibe coded applications by enthusiasts. We need to build up the AI First Development orchestration skills and change with the world to provide value we have always provided, with more automation supporting our own work.

Hours: AI Training. Reapproved reporting expectation for BCS partners.
